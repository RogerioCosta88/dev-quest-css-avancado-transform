Transform

Transform: scale

A propriedade scale é usada para aumentar a escala de um elemento, horizontalmente e verticalmente. Scale Y aumenta o elemento verticalmente e scale X aumenta o elemento horizontalmente.

Transform: rotate

Rotate é a propriedade de gira um elemento, também nos eixos X e Y.

Transform: rotate X

Transform: rotate Y

Transform: translate

O translate move um elemento horizontalmente ou verticalmente.
